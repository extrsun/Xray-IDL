Memo on various Chandra data calibration and analysis procedures
(all files are in the directory /net/xray/pub/cal_scripts)

Basic calibration and analysis procedures:

 
memo_cal - basic calibration, including astrometry correction, evt1 to evt2 processing, clearning, exposure map
                        and construction.

memo_anal - basic IDL data analysis, including source detection, and image construction, smoothing, contour plots, etc.

memo_bg -  get "blank sky" background or stowed  (particle-only) background

memo_csmooth - smooth images with or without sources with the CIAO csmooth routine

bkg_memo  - spectral analysis of diffuse X-ray emission using the double subtraction method (Z. Li)

memo_spec_d - spectral analysis of diffuse X-ray emission

memo_spec_ps  - spectral analysis of a point-like or compact X-ray source

memo_timing - IDL timing analysis

memo_id - find IDs of the sources

memo_merge  - merge events and exposure maps from multiple observations

memo_merge_anal - imaging and spectral analysis of individual features based on a merged event file

memo_sou_det_merged_file - detect sources in merged multiple offset observations


Advanced analysis procedures (testing):

memo_galaxy_sou_plot - plot sources in publishable quality

memo_galaxy_imaging - more hands-on IDL analysis of the imaging data on nearby galaxies.

memo_nfr - source number-flux relationship analysis and residual source contamination determination

memo_nfr_m31  - same as the above, but an example for merged PSF source detection

memo_galaxy_dif  - IDL analysis of source-removed "diffuse" emission
 

Analysis for more specific purposes:
memo_addspec - combine multiple spectra on the same source into one spectrum

memo_line_map - map out the line emission with proper continuum substraction

memo_dstreak_ciao - CCD de-streaking of bright point-like sources, using the standard CIAO method

memo_dstreak - CCD de-streaking of bright extended sources, using the background subtraction method (IDL)

memo_hst_x - overlay X-ray contours on HST images

memo_bg_anal - extraction and analysis of the diffuse X-ray background

memo_galaxy_parameters - estimate of basic parameters of galaxies (mass and SF rate, etc).
